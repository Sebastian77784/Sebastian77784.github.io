1.2 Acciones semánticas de un analizador sintáctico

Las acciones semánticas son operaciones que se ejecutan durante el análisis de la sintaxis de un programa. A diferencia de las acciones sintácticas, que se centran en la estructura gramatical, las acciones semánticas se enfocan en el significado y comportamiento del programa.

Objetivos de las Acciones Semánticas:

  • Indicar errores de forma clara y precisa.
  • Aclarar el tipo de error y su localización.
  • Recuperarse del error para seguir examinando la entrada.
  • No ralentizar significativamente la compilación.

Estrategias de Recuperación de Errores Sintácticos:

  • Recuperación en modo de pánico: Intenta recuperarse después de encontrar un error.
  • Recuperación a nivel de frase: Busca errores en frases completas.
  • Producción de errores: Genera errores cuando se detectan incompatibilidades.
  • Corrección global: Realiza correcciones para continuar el análisis.

Acceso a la tabla de símbolos:

  • Durante el análisis sintáctico, el analizador puede consultar una tabla de símbolos que almacena información sobre variables, funciones y otros símbolos definidos en el programa.
  • Ejemplo: Si se encuentra una variable en una expresión, el analizador verifica si esa variable está declarada previamente en la tabla de símbolos.

Chequeo de tipos:

  • Las acciones semánticas verifican que los tipos de datos involucrados en las expresiones sean compatibles.
  • Ejemplo: Si se suma una variable entera con una cadena de texto, el analizador detectará un error de tipo.

Generación de código intermedio:

  • El analizador puede crear representaciones intermedias del programa para su posterior optimización o traducción.
  • Ejemplo: Convertir una expresión matemática en una representación más abstracta antes de generar código máquina.

Manejo de errores semánticos:

  • Detecta y reporta errores relacionados con la coherencia semántica.
  • Ejemplo: Si se llama a una función con argumentos incorrectos, el analizador generará un error.

Asignación de significado a expresiones:

  • Las acciones semánticas asignan significado a las expresiones y verifican que tengan sentido.
  • Ejemplo: Evaluar una expresión aritmética para obtener un resultado válido.
